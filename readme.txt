欢迎使用易社区（2.1）正式版

简介：
经过很长时间的测试，修正了不少错误与问题，终于可以正式发布了。。。。。
这是一个简单的社区php开源程序，具体可访问 http://what.sinaapp.com 查看。

安装：
1） 首次安装：
解压压缩包后得 wap/,将其中的文件上传至空间，直接访问所上传的目录便可进行配置。 
  注意：安装完成后，请立即注册，id为1的用户默认为超级管理员，在登录后，点击“主页”可看到后台入口。

2）安装过以前版本
先删除以前版本和数据库内相关表单，然后在进行常规安装。
**对于以后版本将不用清除数据库，也就是，网站以前的内容不必清除。
（如需帮助请到 http://what.sinaapp.com 发帖）
